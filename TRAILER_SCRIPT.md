# RESONANCE GRID — OFFICIAL TRAILER SCRIPT
### "FROM THE VOID" — v1.0
**Format:** Gemini Omni video generation (shot-by-shot prompts)
**Target Duration:** ~90 seconds
**Aspect Ratio:** 16:9, 1920×1080
**Visual Style:** Afrofuturist cyberpunk — deep indigo-black backgrounds, neon orange geometry, magenta nebulae, glitch-art transitions, CRT scanline overlays
**Music Tempo:** 132 BPM electronic / tribal fusion — starts as low sub-bass drone, builds to full industrial percussion by midpoint, peaks at final boss reveal

None.

## GEMINI OMNI GENERATION NOTES

When generating each scene, prepend the following style prefix to every visual prompt:

> **Style prefix:** `Afrofuturist cyberpunk game trailer shot. Vector neon art style. Deep indigo-black void background. Neon orange glowing grid geometry. Magenta and purple cosmic nebula atmosphere. CRT scanline overlay. Cinematic bloom lighting. 24fps filmic grain. No photorealism — geometric, graphic, sacred geometry aesthetic.`

**Recommended generation order:**
1. Generate Scenes 01, 11, 12 first — anchor the visual language
2. Generate boss scenes (07, 09) second — set scale references
3. Fill in combat and mission scenes (04, 06, 10)
4. Generate transition scenes (02, 03, 05, 08) last

**Transition prompts between scenes:**
- Smash cut: `Hard cut, no transition, abrupt`
- Glitch wipe: `Diagonal neon orange static wipe, 6 frames`
- Fade to black: `1-second linear fade to pure black`
- White-out: `Overexposed flash fill white, 1 second`

**Text overlay style:**
`Condensed white sans-serif (Bebas Neue / Impact weight). Letter-spacing: wide. Orange accent variant: same font in #FF6A00. All caps. No drop shadow — rely on glow bloom from the background for legibility.`
